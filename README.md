# Faculty Appointment Booking System - Frontend

This is the frontend application for the Faculty Appointment Booking System, built with React and Vite.

## Technology Stack

- **Framework:** React 18
- **Build Tool:** Vite 5
- **Routing:** React Router DOM 6
- **HTTP Client:** Axios
- **State Management:** React Hooks

## Prerequisites

- Node.js (v18.0.0 or higher)
- npm or yarn

## Setup and Running

1. **Navigate to the frontend directory:**
   ```bash
   cd frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## Features

- User Authentication (Login/Register)
- Interactive Faculty Dashboard
- Appointment Booking Calendar
- Responsive Design
